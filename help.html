Welcome to the help page of the Signed Exchange Validator Chrome Extension.

This tool is intended to help website administrators to determine the status of their Signed Exchange deployments. It may include the status of the origin URL, its corresponding Google SXG Cache URL as well as the Google SXG Certificate URL.

The different images the tool displays indicates the status of either the origin, cache or cert.

... means it's processing.
⌛ means not in cache yet.
✅ means everything's ok.
❌ means there's an error.
URL The URL of the SXG's origin, cache or SXG certificate.
Content Type The content type of the URL.
Warning Message

Warning message from the Google Cache server indicating the state of the cache. If this is present, then it indicates an error that prevented the SXG from meeting the cache requirements.
Location The location of the URL. If this is present, then URL has not yet been fetched by the cache. This is not an error in your SXG.
